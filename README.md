# Final Year Project Computing

This repository contains a full-stack healthcare project with a React/Vite frontend and a Node.js/TypeScript backend. It includes authentication, doctor/hospital management, appointment booking, medical records, and Supabase integration.

## Contents

- frontend/: React + Vite application
- backend/: Node.js + TypeScript API with Supabase
- supabase/: database migrations and demo seed data

## Notes

- Add your own environment variables for Supabase and JWT keys
- Run `pnpm install` in both root and backend if needed
- Use `pnpm dev` from the frontend and `pnpm start` from backend if scripts are available
